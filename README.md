# Maze_solving

A reinforcement learning project implementing **Q-learning** to solve a maze environment.  
This repository contains the source code, environment setup, and demonstration files for the assignment.

---

## 📌 Project Structure
. ├── agent.py          # Q-learning agent implementation ├── env.py            # Maze environment definition ├── main.py           # Entry point: training + Pygame demo ├── notebook.ipynb    # Colab/Jupyter notebook with bilingual explanation ├── training.png      # Training curve (generated during training) ├── requirements.txt  # Python dependencies (minimal version) ├── README.md         # Project documentation └── LICENSE           # MIT License
